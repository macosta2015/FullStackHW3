# Media Query 
They are used to adapt to the device. Thye looks for the devices dimensions/characteristsics such as resolution, height, width, etc and start working from there.
We use them when we change the device that we are using. When we open the URL from the phone, the color is different than when we open it from the desktop or tablet.
